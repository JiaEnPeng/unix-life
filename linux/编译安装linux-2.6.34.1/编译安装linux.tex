\section{编译安装linux-2.6.34.1}
	我们要编译安装内核linux-2.6.34.1，这个内核可以在网站\url{https://www.kernel.org/pub/linux/kernel/v2.6/}上下载。\par
	
	首先我们需要编译内核，步骤如下：
	\begin{lstlisting}
	# 配置内核
	sudo apt-get install libncurses5-dev
	make menuconfig

	# 更改编译器
	sudo apt-get install gcc-4.8
	sudo ln -sf /usr/bin/gcc-4.8 /usr/bin/gcc

	# 编译
	make
	\end{lstlisting}

	编译过程中可以遇到的错误：
	\begin{lstlisting}
	1. gcc: error: elf_x86_64: 没有那个文件或目录
	解决方法：在arch/x86/vdso/Makefile中， 
			将以VDSO_LDFLAGS_vdso.lds开头所在行的 "-m elf_x86_64" 替换为 "-m64"。 
			将以 VDSO_LDFLAGS_vdso32.lds 开头所在行的 "-m elf_x86" 替换为 "-m32"。

	2. Can't use 'defined(@array)' (Maybe you should just omit the defined()?) at kernel/timeconst.pl line 373.
	解决方法：将kernel/timeconst.pl中373行的 if (!defined(@val)) 改为 if (!@val) 

	3. drivers/net/igbvf/igbvf.h:129:15: error: duplicate member ‘page’ struct page *page;
	解决方法：将drivers/net/igbvf/igbvf.h文件中第二个struct page *page;注释掉
	\end{lstlisting}

	编译结束后，使用如下命令生成vmlinux：
	\begin{lstlisting}
	sudo make bzImage
	\end{lstlisting}

	随后编译和安装模块，并安装内核：
	\begin{lstlisting}
	make modules
	sudo make modules_install
	sudo make install
	\end{lstlisting}
