\section*{ubuntu配置latex}
	首先声明，这个仅在ubuntu16.04下配置过，配置日期为2017.1.17。\par

	安装latex的命令如下：
	\begin{lstlisting}
	sudo apt-get install texlive-full
	\end{lstlisting}

	到此，ubuntu下的latex就配置好了。我更习惯用visual-studio-code写latex代码，只要在vscode中安装如下两个插件即可：
	\begin{lstlisting}
	LaTex Compile
	LaTex Language Support
	\end{lstlisting}

	ubuntu下安装插件后即可使用，如下图所示：

	\sizedfic{0.6}{2.png}

	根据个人需求，可能还需要更改latex的编译器，vscode中对LatexCompile的配置如下图所示：
	\sizedfic{0.6}{3.png}

	如果想把编译器改成xelatex，只要在settings.json添加如下代码：
	\begin{lstlisting}
	"latexCompile.compiler": "xelatex"
	\end{lstlisting}

	我在mac OS下也配置过latex，操作和上述的流程一样。
