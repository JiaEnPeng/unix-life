\section{使用vncviewer从一台linux远程到另一台linux}
\subsection{客户端linux的配置}
	客户端需要安装vncviewer和vncserver，命令如下：
	\begin{lstlisting}
	sudo apt-get install vnc4server
	sudo apt-get install vncviewer
	\end{lstlisting}

\subsection{服务器端linux的配置}
	服务器端需要安装vncviewer和vncserver，命令如下：
	\begin{lstlisting}
	sudo apt-get install vnc4server
	sudo apt-get install vncviewer
	\end{lstlisting}

	然后开启vncserver的服务：
	\begin{lstlisting}
	vncserver
	\end{lstlisting}

	如下图所示：
	\sizedfic{0.6}{1.png}

\subsection{从客户端linux远程到服务器端linux}
	首先，在服务器端linux开启一个虚拟机：
	\begin{lstlisting}
	virsh vncdisplay instance-000002cf
	\end{lstlisting}

	如下图所示：
	\sizedfic{0.6}{2.png}

	可以注意到，这里的端口是“:1”，这个端口我们之后会用到。\par

	然后，在客户端linux远程到服务器端linux：
	\begin{lstlisting}
	# root是服务器端linux的用户名
	# 172.16.19.239是服务器端linux的IP地址
	ssh -L 5901:localhost:5901 -l root 172.16.19.239
	\end{lstlisting}

	如下图所示：
	\sizedfic{0.6}{3.png}

	然后在客户端linux另外开一个客户端，使用vncviewer远程到服务器端linux上的虚拟机：
	\begin{lstlisting}
	# 端口号1是之前vncdisplay给出的端口号
	vncviewer 172.16.19.239:1
	\end{lstlisting}

	此时就能在客户端linux上看到远程的服务器端linux上的虚拟机：
	\fic{4.png}
